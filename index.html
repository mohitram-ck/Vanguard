// ARCHITECTURE

THE PIPELINE

MODULE 01
DATA INGESTION
scapy · AsyncSniffer · threading
Asynchronous packet capture engine. Sniffs raw network traffic on any interface and pushes structured PacketRecord objects into a thread-safe queue.
  • TCP / UDP / ICMP parsing
  • Thread-safe packet queue
  • Dropped packet tracking
  • Auto interface detection
▶
MODULE 02
FEATURE EXTRACTION
pandas · numpy · rolling window
Converts raw packets into 19 statistical features over a configurable rolling time window. Each vector captures behavioral patterns invisible at the packet level.
  • SYN flood ratio
  • Port scan heuristic
  • Traffic entropy
  • Payload statistics
▶
MODULE 03
AI ENGINE
scikit-learn · Isolation Forest · StandardScaler
Isolation Forest model trained on baseline traffic. Anomalous patterns are isolated in fewer tree splits — attacks score lower than normal traffic.
  • Unsupervised learning
  • 200 estimator trees
  • Persisted model (joblib)
  • Instant on_alert callback
▶
MODULE 04
MITIGATION
iptables · netsh · pfctl · SHA-256
Cross-platform firewall blocking and secure event logging. Every action is written to a hash-chained audit log — tampering breaks the chain.
  • IP auto-block
  • IP whitelist safety
  • Hash-chained audit log
  • Dry-run dev mode

// SECURITY DESIGN

BUILT WITH SECURITY IN MIND

🔗
HASH CHAIN AUDIT LOG
Every event is logged in a JSONL file where each entry contains the SHA-256 hash of the previous one — forming a tamper-evident chain. Any modification breaks all subsequent hashes.
🧱
CROSS-PLATFORM FIREWALL
Automatically selects the correct firewall backend: iptables on Linux, netsh on Windows, pfctl on macOS. Zero manual config required.
🧵
THREAD-SAFE PIPELINE
All 4 modules run in isolated background threads. Communication happens exclusively through bounded, thread-safe queues — no shared mutable state, no race conditions.
🎓
UNSUPERVISED ML
Isolation Forest requires no labeled attack data. It learns what "normal" looks like from baseline traffic, then flags statistically anomalous deviations — detecting zero-day attack patterns.
🛡️
IP WHITELIST SAFETY
Configurable IP whitelist prevents the system from accidentally blocking its own host, gateway, or trusted infrastructure — a critical safeguard in automated response systems.
🧪
DRY RUN MODE
Firewall commands are logged but never executed in dry-run mode. Allows safe development, testing, and demonstration on any machine without admin rights or network risk.

from scapy.all import AsyncSniffer, IP, TCP, UDP
from dataclasses import dataclass

@dataclass
class PacketRecord:
    """Structured representation of a captured packet."""
    timestamp: datetime
    src_ip:    str
    dst_ip:    str
    protocol:  str
    payload_size: int
    flags:     Optional[str]

class PacketCaptureEngine:
    def __init__(self, interface=None, queue_maxsize=10000):
        self.packet_queue = queue.Queue(maxsize=queue_maxsize)
        self._sniffer     = None
        self._running     = False
        self._captured    = 0

    def start(self):
        self._sniffer = AsyncSniffer(
            iface=self.interface,
            prn=self._handle_packet,
            store=False   # Don't buffer in memory
        )
        self._sniffer.start()
        logger.info("✅ Capture engine started")

class FeatureExtractor:
    def _compute_features(self) -> Optional[dict]:
        df = pd.DataFrame([...])  # Buffer → DataFrame

        # ── SYN Flood Detection ─────────────────
        syn_packets = df[
            df["flags"].str.contains("S", na=False) &
            ~df["flags"].str.contains("A", na=False)
        ]
        syn_ratio = len(syn_packets) / total_packets

        # ── Traffic Entropy ──────────────────────
        src_ip_counts = df["src_ip"].value_counts(normalize=True)
        entropy = -np.sum(
            src_ip_counts * np.log2(src_ip_counts + 1e-9)
        )

        # ── Port Scan Heuristic ──────────────────
        ports_per_src = df.groupby("src_ip")["dst_port"].nunique()
        max_ports_per_src = float(ports_per_src.max())

        return {"syn_ratio": syn_ratio, "src_ip_entropy": entropy, ...}
# Vanguard NIDS — AI Engine Module
class AnomalyDetector:
    def train(self, training_data: list[dict]) -> None:
        X = np.array([[d.get(k, 0.0) for k in FEATURE_KEYS]
                       for d in training_data])

        self._scaler = StandardScaler()
        X_scaled = self._scaler.fit_transform(X)

        self._model = IsolationForest(
            n_estimators=200,    # More trees = stability
            contamination=0.05,  # 5% expected anomaly rate
            random_state=42,
            n_jobs=-1            # All CPU cores
        )
        self._model.fit(X_scaled)
        joblib.dump(self._model, MODEL_PATH)

    def predict(self, features: dict) -> DetectionResult:
        X_scaled = self._scaler.transform(self._vectorize(features))
        raw_pred  = self._model.predict(X_scaled)[0]  # +1 or -1
        label = "BENIGN" if raw_pred == 1 else "MALICIOUS"
        return DetectionResult(label=label, ...)
# Vanguard NIDS — Mitigation Engine Module
class SecureAuditLogger:
    def log(self, event_type: str, data: dict) -> None:
        with self._lock:
            entry = {
                "timestamp":  datetime.utcnow().isoformat(),
                "event_type": event_type,
                "data":       data,
                "prev_hash":  self._last_hash,  # Hash chain
            }
            entry["entry_hash"] = self._hash_entry(entry)
            self._last_hash = entry["entry_hash"]

class FirewallManager:
    def block_ip(self, ip: str) -> bool:
        if IS_LINUX:
            cmd = ["iptables", "-A", "INPUT", "-s", ip, "-j", "DROP"]
        elif IS_WINDOWS:
            cmd = ["netsh", "advfirewall", "firewall", "add", "rule",
                   f"name=Vanguard_Block_{ip}", "action=block", ...]
        elif IS_MAC:
            cmd = ["pfctl", "-t", "vanguard_blocklist", "-T", "add", ip]
        self._run_command(cmd)
